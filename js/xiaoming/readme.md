## js 面向对象
- JS的数据类型
{} 对象 Object
[] 数组 
字符串 "叶玉婷" String
数值型 Number 3
布尔型 Boolean
- JS中面向对象有2种表达方式
 1. class 方式来构造抽象的类,以及可使用的对象的概念,通过 constructor 类型和数量(type,n)
 2. 对象字面量的方式来构建 { } 描述性 属性+方法的复杂数据结构
 eg: yezong.sendflower(xiaoming);
- 代理模式 Proxy(面向接口的编程)
eg: 婷总 ->小明 (简陋的面向对象)
eg: 婷总 ->小美 ->小明  
不同的对象(小明,小美) ,只要实现同样的方法
(receiveFlower),交换使用,面向接口的编程
