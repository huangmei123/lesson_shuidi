1. 暴力法 简单的括号算法

2. 将时间复杂度降低
tmpMax 来计算,左右括号 下标之间做减法 得出长度
())(())
一次遍历 i 下标法 0 
-1 哨兵元素 第一个元素进入 0-(-1)
栈为空 提前退出 0前面的匹配的长度
省循环的根本 重新再来
